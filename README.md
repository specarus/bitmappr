# BitMappr

## Technologies

- **React 18** for composable UI logic and stateful interactions.
- **Vite 5** as the performant dev server/bundler with zero-config module handling.
- **p5.js** (wrapped in `src/sketch.js`) for generative drawing and animation control.
- **lucide-react** for crisp, scalable iconography across controls or overlays.
- **ESLint + React plugins** to keep code consistent, accessible, and predictable.

## Setup Instructions

1. **Prerequisites**
   - Node.js 18 LTS (or later) and npm installed.
2. **Install dependencies**
   ```bash
   npm install
   ```
3. **Develop locally**
   ```bash
   npm run dev
   ```
